## setup

1. run these
    ```bash
    git clone https://github.com/yuvlian/sebbye.git
    cd sebbye
    npm install
    npm run build
    ```
2. open `chrome://extensions` on ur browser.
3. enable developer mode and click load unpacked.
4. select the `dist` dir from build result

## usage guide

1. open ur moodle quiz page or whatever that requires seb
2. download the `config.seb`
3. if the config is encrypted, uhh, i dont have a decryptor for it yet, sorry
4. in the sebbye extension popup: click "choose seb config" and select ur `config.seb`
5. refresh
6. configure fake bars as needed
7. fullscreen your page to hide ur browser by pressing `f11` or `fn+f11` if that didnt work
